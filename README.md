## Запуск
sudo python3 main.py
- можно установить с какого хоста запускать **-host '...'** . По умолчанию 127.0.0.1
- можно установить с какого порта запускать  **-p ** . По умолчанию 5005

## Пример для dig
- dig yandex.ru @127.0.0.1 -p 5005
